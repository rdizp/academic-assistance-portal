# Academic Assistance Portal

Academic Assistance Portal is a web-based system developed for the Laboratory of FISIP UNIGA to monitor student assistance progress and seminar eligibility requirements.

## Features

* Real-time assistance tracking
* Seminar eligibility monitoring
* Assistance validation status
* Summary Card generation
* Mobile-friendly interface

## Technology Stack

* Google Apps Script (Backend API)
* Google Sheets (Database)
* HTML, CSS, JavaScript (Frontend)
* GitHub Pages (Hosting)
